# Final Fantasy VII Remake – Characters API

Aplicación desarrollada en **Node.js + Express + Pug** para gestionar personajes de la trilogía *Final Fantasy VII Remake*.  
Incluye una **API REST completa**, **vistas en Pug** y una **colección Postman** con pruebas automatizadas.

---

## Tecnologías utilizadas

- **Node.js**
- **Express**
- **Pug**
- **JavaScript**
- **Postman**

---

## Instalación y ejecución

1. Clona el repositorio:

```
git clone https://github.com/Guille33-dev/ActividadFinalFantasy.git
```
2. Instala dependencias:
```
npm install
```

3. Inicia el servidor:
```
npm start
```

4. Accede en el navegador:
```
http://localhost:8080/index
```
