Final Fantasy VII Remake – Characters API

Aplicación desarrollada en Node.js + Express + Pug para la gestión de personajes de la trilogía Final Fantasy VII Remake.
Incluye API REST completa, vistas Pug y colección Postman para pruebas automatizadas.

Tecnologías utilizadas

Node.js

Express

Pug

Postman (colección de pruebas)

JavaScript

Instalación y ejecución

Clona el repositorio:

git clone https://github.com/Guille33-dev/ActividadFinalFantasy.git


Instala dependencias:

npm install


Inicia el servidor:

npm start


Accede a la aplicación:

http://localhost:8080/index
